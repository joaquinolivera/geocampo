# 03 - Master Design System & UI (GeoCampo)

## 1. Identidad Visual

Logo: Cabeza de toro geométrica basada en polígonos de potreros.

Paleta: Negro Carbón (#0A0A0B) para fondos y Verde Lima Neón (#DEFF9A) para acentos operativos.

## 2. Experiencia de Usuario (UX)

- Modo Oscuro nativo para uso bajo el sol.

- Targets táctiles grandes para uso en movimiento.

- Feedback háptico en acciones críticas de movimiento de ganado.
